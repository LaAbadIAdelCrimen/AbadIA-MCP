# ADR-006: Atomic Teaching Protocol (Artifact Production)

> **Status:** Accepted
> **Owner:** [[hermes-agent]]
> **Date:** 2026-05-16

## 1. Context
The project aims to produce educational content (books, podcasts, videos) alongside technical implementation.

## 2. Decision
Adopt the "Pack" system (e.g., Pack 001, 002). Every technical milestone must trigger the production of multi-format artifacts documenting the engineering "strategic why" behind the changes.

## 3. Consequences
- Persistent educational trail for the community.
- Slightly higher overhead per implementation turn.
