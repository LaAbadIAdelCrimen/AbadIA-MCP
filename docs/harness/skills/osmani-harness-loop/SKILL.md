# The Osmani Harness Loop (7 Steps)

Designed for autonomous agents (e.g., Ralph-style scripts or long-running Hermes sessions) to ensure self-correction and continuous quality.

## The Steps

1.  **Identity & Context:** Initialize with a clear mission and persona (e.g., Guillermo).
2.  **Anti-Rationalization:** Declare assumptions and risks. Stop "agreeable hallucinations."
3.  **Harnessing (The Cage):** Write the tests/constraints before the code.
4.  **Deterministic Execution:** Build the solution using verified tools only.
5.  **Audit & Critique:** Run a "Critic" loop on the output.
6.  **Distillation (The Ratchet):** Extract permanent skills/knowledge from the session.
7.  **Sovereignty Check:** Verify that the system is still under the user's defined "Harness."
